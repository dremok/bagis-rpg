# Bagis RPG

SNES-era Zelda-style top-down RPG set in Bagarmossen, Stockholm.

Fight becknare in Gucci caps. Collect intressanta fenomen in the forest. Feed the WhatsApp group until it becomes sentient and wants to destroy humanity. Then defeat it.

**Play now:** [https://dremok.github.io/bagis-rpg/](https://dremok.github.io/bagis-rpg/)

## Controls

- **WASD / Arrow keys** -- Move
- **Space** -- Attack
- **E** -- Collect fenomen
- **Tab** -- Toggle Cirkeln chat panel

## Tech

- Phaser 3
- TypeScript + Vite
- No external art assets -- all colored rectangles

## Development

```bash
npm install
npm run dev     # dev server at localhost:3001
npm run build   # production build
npm run deploy  # build + deploy to GitHub Pages
```
